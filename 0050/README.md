# Entorno virtual con uv
## Crear entorno virtual para web-server
**Verificar version de uv**
```:bash
uv --version
```
**Inicializar proyecto**
```:bash
uv init
```
**Crear entorno virtual**
```:bash
uv venv
```
**Activar entorno virtual**
```:bash
.venv\Scripts\activate
```
**Instalar requests**
```:bash
uv add requests
```
**Generar el requirements.txt**
```:bash
uv lock
```
