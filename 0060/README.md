## Crear entorno virtual para `python-testing`
**Verificar version de uv**
```:bash
uv --version
```
**Inicializar proyecto**
```:bash
uv init
```
**Crear entorno virtual**
```:bash
uv venv
```
**Activar entorno virtual**
```:bash
.venv\Scripts\activate
```
**Generar el requirements.txt**
```:bash
uv lock
```
